Areas

Area Class

Area classes are generated using yaml configuration files. To make their usage easier, lcls-tools has a function for generating configured Areas. Please see the example usage below:

 1"""Example usage for creating areas """
 2from lcls_tools.common.devices.reader import create_area
 3# Generate area for L1B
 4L1B = create_area('L1B')
 5
 6# After creation, we can access devices collections and print their names
 7# and perform group operations on subsets of the area devices
 8magnet_collection = L1B.magnet_collection
 9print(magnet_collection.device_names)
10
11# If we want to directly access individual devices, we can do this:
12magnets = L1B.magnets
13print(magnets['QCM02'].name)
14print(magnets['QCM02'].bdes)
15print(magnets['QCM02'].bact)
16
17# After creation, we can access devices collections and print their names
18# and perform group operations on subsets of the area devices
19DOG = create_area('DOG')
20screen_collection = DOG.screen_collection
21print(screen_collection.device_names)
22
23# If we want to directly access individual devices, we can do this:
24screens = DOG.screens
25print('Name: ', screens['OTRDOG'].name)
26print(f'Dimensions: ({screens['OTRDOG'].n_rows}, {screens['OTRDOG'].n_cols}).')
27print(f'Resolution: {screens['OTRDOG'].resolution)
class lcls_tools.common.devices.area.Area(name, *args, magnets: Annotated[MagnetCollection, SerializeAsAny()] | None = None, screens: Annotated[ScreenCollection, SerializeAsAny()] | None = None, wires: Annotated[WireCollection, SerializeAsAny()] | None = None, bpms: Annotated[BPMCollection, SerializeAsAny()] | None = None, lblms: Annotated[LBLMCollection, SerializeAsAny()] | None = None, pmts: Annotated[PMTCollection, SerializeAsAny()] | None = None)

This class provides access to collections of hardware components in a given machine area of LCLS/LCLS-II (for example: BC1, or BC2). The information for each collection is provided in YAML configuration files, where the filename is the machine area.

Variables:

  • magnet_collection – The MagnetCollection for this area

  • screen_collection – The ScreenCollection for this area

property bpms: Dict[str, BPM] | None

A Dict[str, BPM] for this area, where the dict keys are bpm names If no bpms exist for this area, this property is None

property lblms: Dict[str, LBLM] | None

A Dict[str, LBLM] for this area, where the dict keys are lblm names If no lblms exist for this area, this property is None

property magnets: Dict[str, Magnet] | None

A Dict[str, Magnet] for this area, where the dict keys are magnet names. If no magnets exist for this area, this property is None.

property pmts: Dict[str, PMT] | None

A Dict[str, PMT] for this area, where the dict keys are pmt names If no pmts exist for this area, this property is None

property screens: Dict[str, Screen] | None

A Dict[str, Screen] for this area, where the dict keys are screen names If no screens exist for this area, this property is None.

property wires: Dict[str, Wire] | None

A Dict[str, Wire] for this area, where the dict keys are wire names If no wires exist for this area, this property is None
